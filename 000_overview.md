Here is the full final curriculum for a Rust basics course tailored for experienced programmers in Python and C, translated into English:

### Rust Basics Course for Experienced Programmers

| Module                     | Contents                                                      |
|----------------------------|---------------------------------------------------------------|
| Introduction & Motivation   | Rust principles, key differences compared to C and Python     |
| Tooling & Environment       | Installation, rustup, Cargo, project structure, rustdoc       |
| Getting Started             | Syntax, variables, typing, comparison to C & Python           |
| Data Structures            | Primitive types, structs, enums, pattern matching             |
| Functions                  | Function definitions, closures, ownership in function params  |
| Ownership & Borrowing      | Ownership, borrowing, lifetimes, common errors                 |
| Error Handling             | Result, Option, panic, idiomatic error management              |
| Modularization & Traits    | Modules, packages, crates, traits, code reuse                  |
| Advanced Types             | Generics, operator overloading, standard traits                |
| Iteration                  | Iterators, combinators, vectors, hash maps and sets            |
| Multithreading & Async     | Concurrency, threads, async/await, channels, Tokio runtime     |
| FFI & Interoperability     | Interaction with C/C++, bindings, overview                      |
| **Macros in Rust**         | Declarative macros (`macro_rules!`), procedural macros, examples, differences to C macros, common pitfalls |
| Testing & Documentation    | Unit tests, rustdoc, best practices                             |
| Practical Project          | Own Rust project: CLI tool, simple API, or application         |
| Advanced Topics            | Rust ecosystem, important libraries, community, further steps  |

This structure offers a deep, practical introduction to Rust while emphasizing the differences and similarities to C and Python, with a dedicated module on macros as an advanced but essential part of Rust programming.
